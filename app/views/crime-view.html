Seattle Crime Reports

The following examples all use data that can be found at data.seattle.gov. An HTTP requests a JSON object at page load. JSON data is then processed using D3 and native JavaScript, and finally rendered as Angular directives. All visualastions are SVG elements generated by D3. Mapping is done with Google Maps JavaScript API.

Coded by Kyle Moseby.

A GitHub Repo can be found here

Dependencies

  • AngularJS
  • D3
  • Google Maps
